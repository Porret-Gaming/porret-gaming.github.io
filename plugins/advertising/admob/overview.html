Advanced Admob Ads

Advanced Admob Ads

The Google Admob Library is a tool used to display various different types of adverts within your mobile application. With this, developers can monetize their games by displaying advertisements in accordance with their guidelines.

  • Privacy Settings (GDPR and COPPA)
  • Banner Ads
  • App Open Ads
  • Interstitial Ads
  • Rewarded Interstitial Ads
  • Rewarded Video Ads
  • Mediation

By making use of these different types of advertisements within your appication at key points, you can unlock additional revenue streams, ultimately creating a sustainable business model for your Android Game.

Google Play Games Services:

Using this plugin, you will no longer be able to make use of the Unreal Engine provided Google Play Game Services.

In the event you make use of these services, I have released a Seperate Plugin which upgrades the built-in Google Play Games libraries to be compatible. This seperate plugin will replace and drastically improve all features of the default Play Games Services provided by Unreal Engine.

I have managed to find a compatible library version of the Google Play Games libraries which can work with the latest Admob libraries and the built-in Unreal Engine Google Play Game functionality. This plugin now updates to the latest version of the library that can support the built-in Google Play Game services as well as be compatible with the Admob Library.

Please note, as I update this plugin's admob library, this may change as the versions get further and further apart. It is still strongly encouraged that you consider the Play Base Services V2 plugin which will always be compatible and provide a much more enhanced Google Play Games service integration.

The Unreal Engine Plugin

Introducing the Unreal Engine plugin for the Admob service, a powerful tool that brings all the features of the latest Google Admob Service directly to your Unreal Engine projects. This plugin is designed to be as simple as possible, while exposing as much of the library features to you.

We have taken special measures to handle all Engine versions from Unreal Engine 4.27 onwards to cater for all the requirements of keeping the Admob library as up to date as possible. With the latest version of Admob, the outdated UE built in Google Play Services library may no longer work. We have also released a plugin which offers the upgraded Google Play Services to allow you to use both in the event that is part of your requirements. In addition to this, all my available plugins can automatically detect this upgrade and adapt to the latest library as well.

As stated previously I have managed to find a compatible google play games library version which allows you to use existing google play game features offered by Unreal Engine. Once again, Please note that this may not remain like this as the library versions get further and further apart, and I still strongly encourage you to consider using my Play Base Services V2 plugin.
