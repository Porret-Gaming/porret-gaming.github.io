Getting Started

  1. Create a PNG

    2. Create your PNG image in whichever way you see fit.

  2. Import PNG into Unreal Engine

    3. Drag or import your image into the Unreal Engine Editor, ultimately creating a texture object.

  3. Open Texture Settings

    4. Double click your recently created Unreal Engine texture object to open a new tab showing the textures settings.

  4. Update Settings

    5. Within the texture settings for this texture object, update the:

    Compression Settings to VectorDisplacementmap (RGBA8)
    Ensure sRGB is set to false
    Mip Gen Settings to NoMipmaps

    You can now use this texture as an input field for our blueprint nodes.
