Setup Sensor Event Listener

This function sets up the delegates for when registered sensors are triggered or their state is changed.

The function takes two delegate parameters, "On Sensor Accuracy Changed" and "On Sensor Value Changed".
This blueprint node can be executed in multiple places across your project and all delegates will be triggered.

1

On Sensor Accuracy Changed

A delegate called when a sensor's accuracy has changed, normally due to a change in state (eg. pausing and resuming).

2

On Sensor Value Changed

A delegate called when a sensor reports a change in value.
The Sensor Delay used when Registering a Sensor determines how often the "On Sensor Value Changed" delegate will be called.
