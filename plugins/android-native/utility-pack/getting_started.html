Getting Started

  1. Download the Plugin

    2. Download the plugin on the Official Unreal Engine Marketplace (Disabled until go live).

  2. Install the Plugin

    3. In the Epic Games Launcher, the plugin will appear in your Library.
    Go to the Library tab and find the plugin and add it to the engine.

  3. Enable Android Utility Pack Plugin

    4. In Unreal Engine, the plugin will be available in the plugins section.
    Go to Settings -> Plugins -> Search for "Android Utility Pack" and ensure it is enabled.
