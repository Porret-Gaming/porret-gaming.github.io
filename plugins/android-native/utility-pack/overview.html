Android Utility Pack Overview

Android Native: Utility Pack

Introducing the FREE Android Utility Pack, a comprehensive Unreal Engine plugin designed to enhance your Android mobile application's functionality and user experience. This versatile utility pack empowers developers with a suite of essential tools.

With this Utility Pack, you can seamlessly integrate these critical utilities into your application, ensuring smooth performance and robust functionality. Effortlessly monitor and log application behaviour, managing situations that would otherwise be impossible.

This plugin serves as a repository for smaller, yet crucial functionalities that might not warrant standalone plugins, ensuring your development process remains streamlined and efficient.

If you have ideas that align with this plugin's capabilities, we'd love to hear from you! Join our Discord Server and share your suggestions. Your idea might just become part of the this plugin's suite of features.

The Unreal Engine Plugin

The Unreal Engine plugin for the Mobile Utility Pack is an easy-to-use tool with all its functionality exposed to Unreal Engine blueprints. Much of its functionality can be configured within the Project Settings, providing you with complete flexibility in how it operates.

Some key features include:

Automatic Android UE Source Code Fixes
Android ADB Logging
Internet Connectivity Checks
Android Lifecycle Events
Android Battery Information
SDK Version Detection
