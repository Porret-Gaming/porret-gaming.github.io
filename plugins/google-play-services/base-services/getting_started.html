Getting Started

  1. Purchase the Plugin

    2. Purchase the plugin on the Official Unreal Engine Marketplace.

  2. Install the Plugin

    3. In the Epic Games Launcher, the plugin will appear in your Library.
    Go to the Library tab and find the plugin and add it to the engine.

  3. Enable Play Base Services Plugin

    4. In Unreal Engine, the plugin will be available in the plugins section.
    Go to Settings -> Plugins -> Search for "Play Base Services" and ensure it is enabled.

  4. Enable Version 2

    5. Go to the Plugins Configuration in Project Settings -> Plugins -> Google Play Base V2 and tick Enable Google Play Services V2.

    This toggle allows this plugin as well as our other plugins (Saved Games and Events) to detect the upgrade and adapt appropriately.
